> Use this page to find which commands are available to Staff, Admin, Management, and server administrators.

Commands may also be available via prefix (default **`w!`**) depending on your server setup. This list focuses on slash commands.

<Note>
This list may not include every command. Run **`/help`** in your server for the most up-to-date list.
</Note>

## Server administrator

These commands require **Server Administrator** permission in Discord:

- **`/config`** ‚Äî Open the module configuration panel
- **`/prefix`** ‚Äî Change the bot prefix for your server
- **`/forceverify`** ‚Äî Force-verify a linked user and apply verified roles

## Management role

These commands require a configured **Management** role (or Server Administrator):

- **`/session start`** ‚Äî Start a roleplay session
- **`/session vote`** ‚Äî Create a session start vote
- **`/session end`** ‚Äî End the current session
- **`/session full`** ‚Äî Send the server full notification
- **`/session boost`** ‚Äî Send the space-available notification
- **`/infraction issue`** ‚Äî Issue a staff infraction
- **`/infraction list`** ‚Äî View a staff member's infractions
- **`/infraction manage`** ‚Äî Manage a specific infraction by ID
- **`/erlc link`** ‚Äî Link your ER:LC API key
- **`/erlc unlink`** ‚Äî Unlink your ER:LC API key

## Admin role

These commands require a configured **Admin** role (or higher):

- **`/punishment manage`** ‚Äî Manage an existing in-game punishment by ID
- **`/shift admin`** ‚Äî Manage another staff member's shifts
- **`/shift quota reset`** ‚Äî Reset shift quota data for a shift type
- **`/shift quota status`** ‚Äî View quota status for all staff

## Staff role

These commands require a configured **Staff** role (or higher):

- **`/punish`** ‚Äî Issue a warn, kick, ban, or ban BOLO in-game
- **`/punishments view`** ‚Äî View punishments for a Roblox username
- **`/bolo active`** ‚Äî View all active BOLOs
- **`/shift manage`** ‚Äî Clock in, take a break, or clock out
- **`/shift time`** ‚Äî View shift time for yourself or another member
- **`/shift active`** ‚Äî View who is currently on duty
- **`/shift leaderboard`** ‚Äî View the shift leaderboard
- **`/application pending`** ‚Äî View pending application submissions
- **`/application manage`** ‚Äî View a member's application history

## All members

These commands are available to any server member (when the relevant module is enabled):

- **`/verify`** ‚Äî Link and verify your Roblox account
- **`/unverify`** ‚Äî Remove your verification from the server
- **`/apply`** ‚Äî Submit an application
- **`/feedback submit`** ‚Äî Submit staff feedback with a rating
- **`/feedback list`** ‚Äî View feedback for a staff member
- **`/whois`** ‚Äî Look up Discord or Roblox account info
- **`/help`** ‚Äî View available commands
- **`/ping`** ‚Äî Check Warden's latency
- **`/invite`** ‚Äî Get Warden's invite link

## ER:LC commands

Available when your server is linked via **`/erlc link`**. Most require Staff role or higher:

| Command | Description |
| --- | --- |
| **`/erlc info`** | View in-game server information |
| **`/erlc players`** | List players currently in-game |
| **`/erlc execute`** | Run an in-game command (for example, `:m message`) |
| **`/erlc check`** | Check if in-game players are in the Discord server |
| **`/erlc logs`** | View recent command logs |
| **`/erlc kills`** | View recent kill logs |
| **`/erlc joins`** | View recent join logs |
| **`/erlc bans`** | Search server bans |
| **`/erlc modcalls`** | View recent modcalls |
| **`/erlc vehicles`** | List spawned vehicles |
| **`/erlc team`** | View players on a specific team |
| **`/erlc queue`** | View the player queue |
| **`/erlc calls`** | View recent 911 calls |
| **`/erlc player`** | View detailed info about a player |
