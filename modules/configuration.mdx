---
title: "Configuration"
description: "Open Warden's configuration panel and configure every module from Discord."
---

Warden is configured entirely in Discord — no config files or dashboard required. Use the **`/config`** slash command or the prefix command **`w!config`** (aliases: `cfg`, `configuration`, `setup`) to open the configuration panel.

<Note>
You need **Server Administrator** permission to run the config command.
</Note>

## Open the configuration panel

<Steps>
  <Step title="Run the command">
    Type **`/config`** or **`w!config`** in any channel Warden can see.
  </Step>
  <Step title="Select a module">
    Use the **Select a module to configure...** dropdown to pick the module you want to set up.
  </Step>
  <Step title="Save your changes">
    Most settings save automatically when you interact with menus and buttons. Some flows (like shift types or log messages) have explicit **Save** buttons — click those when you're done.
  </Step>
</Steps>

## Available modules

| Module | Description |
| --- | --- |
| [Permissions](/modules/permissions) | Staff, Admin, and Management roles used across Warden |
| [Roblox Moderation](/modules/roblox-moderation) | Punish players in your ER:LC server from Discord |
| [Verification](/modules/verification) | Link Discord accounts to Roblox profiles |
| [Shift Logging](/modules/shift-logging) | Track staff shift hours and on-duty status |
| [ER:LC Server Logs](/modules/erlc-logs) | Real-time join, leave, kill, modcall, and command logs |
| [Sessions](/modules/sessions) | Start, poll, and announce roleplay sessions |
| [Infractions](/modules/infractions) | Issue and track staff disciplinary actions |
| [Staff Feedback](/modules/staff-feedback) | Let members submit feedback on staff |
| [Applications](/modules/applications) | Custom application forms with review workflows |
