---
title: "Roblox Moderation"
description: "Configure in-game moderation, logging, and player notifications."
---

The **Roblox Moderation** module lets your staff punish rule-breakers directly in your ER:LC private server from Discord.

## How to configure

1. Run **`/config`** or **`w!config`**
2. Select **Roblox Moderation**
3. Enable the module and set your logging channel
4. Optionally configure **Notify Users on Moderation** under **Configure other options...**

## Settings

| Setting | Description |
| --- | --- |
| **Module State** | Enable or disable the module. |
| **Logging Channel** | Text channel where moderation actions are logged. |
| **Notify Users on Moderation** | When enabled, players receive an **in-game notification** when they are warned through Warden's ER:LC API integration. |

<Tip>
Set up your ER:LC API connection before using moderation. Warden needs API access to issue punishments in-game.
</Tip>
