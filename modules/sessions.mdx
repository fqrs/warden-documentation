---
title: "Sessions"
description: "Poll, launch, and announce ER:LC roleplay sessions from Discord."
---

Sessions turn "who wants to hop on?" into structured votes and polished announcements. Management runs the commands; members see embeds in your chosen channel with optional role pings.

<Note>
  Requires [ER:LC Integration](/configuration/erlc-integration) so Warden can read server state and post join details.
</Note>

## Management commands

| Command | Use when… |
| --- | --- |
| **`/session vote`** | You want members to vote before starting |
| **`/session start`** | You are ready to open the server now |
| **`/session end`** | The session is over |
| **`/session full`** | The in-game server hit capacity |
| **`/session boost`** | Slots freed up on a full server |

## Core settings

| Setting | Purpose |
| --- | --- |
| **Module State** | Enable session tooling |
| **Sessions Channel** | Where announcements land |
| **Ping Roles** | Roles mentioned on start (up to 10) |

Configure under **`/config`** → **Sessions**.

## Session settings

Open **Configure other options...** → **Session Settings**:

| Option | Behavior |
| --- | --- |
| **Auto Start** | Starts automatically when a poll hits the vote threshold |
| **Mention Voters** | Pings everyone who voted when the session opens |
| **Ping @here** | Adds `@here` to start announcements |
| **Ping @everyone** | Adds `@everyone` — use sparingly |

## Customize embeds

Open **Configure other options...** → **Customize Embeds** to edit:

- **Start Embed** — Session is live
- **Shutdown Embed** — Session ended
- **Poll Embed** — Vote posted
- **Full Embed** — Server full notice
- **Boost Embed** — Space available again

Each supports color, title, description, thumbnail, and image. Placeholders: [Variables](/configuration/variables).
