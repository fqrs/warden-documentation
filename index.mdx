> Warden is a Discord management bot built to simplify staff operations, moderation workflows, and automation for ER:LC communities.

**Warden** helps server owners and administrators stay organized by centralizing tools like in-game moderation, shift logging, session management, infractions, applications, and live ER:LC logs — all configurable in just a few clicks.

## What can you do with Warden?

- **Punish** players for breaking server rules directly from Discord
- Track staff _performance_ and activity with **Shift Logging**
- Stay _informed_ with real-time **In-Game Logs** — command logs, join/leave logs, modcall logs, and kill logs
- Manage your **Sessions** with start votes, announcements, and automation
- Manage **Staff Infractions** with approval flows and custom embeds
- Collect **Staff Feedback** and run custom **Applications**
